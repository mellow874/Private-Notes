Private Notes app, built using ReactJS, Tailwindcss, Nodejs, Express and Supabase. 
The private-notes app has got the sign-up page, sign-in page, dashboard page which consists of the sidebar, which has to user-profile, logout button, the create file button, then next to the sidebar is the main page, with the searchbar, and note cards that you will create. 
Upon opening it the app, users have to first create their own account, it will require your email and enter a password you won't forget. 
You will then proceed to logging in, using the same details you used to create the account.
Upon logging in you can start creating your notes and saving them, and it you now have too many files, you can use the searchbar to search for the file using its unique title.
